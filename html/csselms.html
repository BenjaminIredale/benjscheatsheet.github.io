CSS Element Positioning

How elements in CSS can snuggle up to, or snub, each other

Tag Name Function
<z-index> CSS z-Index Property This property specifies which elements will be at the fore, and which in the back, when more than one overlap. The element with the highest z-index integer will be further forward.
<display> CSS Display Property This property assignbs the 'render block' of an element. The three most common values for this property are <block>, <inline>, and >inline-block<. Block-level elements take up the full width of their container with line breaks before and after, and can have their height and width dimensions adjusted manually. Inline elements take up as little space as possible, flow horizontally, and cannot have their width or height manually adjusted. Inline-block elements can appear next to each other, AND can have their width and height manually adjusted.
<fixed> Fixed CSS Positioning The default positioning for an element is <static>. However, this property can be assigned different values. When the CSS position for an HTML element is set to <position: fixed>, it is set/pinned to a specific spot on the page(screen), even after scrolling.
<absolute> Absolute CSS Positioning The <absolute> value for the CSS position property lets the element ignore sibling elements and instead be positioned relative to its closest parent element that is positioned with <relative> or <absolute> <absolute> removes an element entirely from the document flow. The positioning attributes <top>, <left>, <bottom>, and <right>, allow the element to be positioned anywhere.
<relative> Relative CSS Positioning This value of the CSS position property allows the element to be positioned anywhere relative to its deafult position on the web page. The offset properties can be used to set the actual position of the element, relative to the original. Without offset properties, the positioning property would act as though it has the default <static> property.
<float> CSS Float Property This property determines how far left or right an element should float within its parent element. The value <left> floats an element to the left side of its container, and similarly for <right>. The width of the container must be specified, else the element will assume the full wdith of its containing element.
<clear> CSS Clear Property This property specifies how an element should behave when it bumps into another element within the same containing element. This is ususally used in combination with elements having the CSS <float> property. This determines on which sides floating elements are allowed to float.
