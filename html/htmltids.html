HMTL Tag Tidbits

A salmagundi of specific HTML tags, and their purposes

Tag Name Function
<em> Emphasis Element When these tags are placed around a text, the text will be emphasised. This will usually manifest in italics by default, for most browsers.
<br> Line Break Element This tag will create a line break in the text in the browser, which is especially useful for visual formatting, or for purposes such as poems or addresses.
alt alt Attribute Used within the <img> tag, this will describe an image, should the image fail to load for whatever reason. If screen reading software is used for the visually impaired, the alt attribute will be read aloud as the text descriptor for the image.
<span> Span Element An inline container for text, which can be used to group a particular section for styling. Only really useful as a generic container, since it lacks specficity for one particular example.
<strong> Strong Element When these tags are placed around a text, the text will be highlighted. This will usually manifest in bold, for most browsers.
<a> Anchor Element This is used to create hyperlinks in the HTML document, whether pointing to other webpages or files on the same server, a location on the same page, or to any other URL via the hyperlink reference href.
<target> Target Attribute Used on the <a> anchor, this specifies where a hyperlink should be opened. The value of "blank" will make the hyperlink to open in a new tab in modern browsers, or a new window in older browsers or when a modern browser has its setting arranged so.
